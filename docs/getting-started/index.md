# Getting Started

Choose how you want to run the platform:

| Option | Best for |
|---|---|
| [Local Development](local-dev.md) | Development, debugging, rapid iteration |
| [Docker](docker.md) | Full-stack testing, production-like environment |

---

## Prerequisites

| Requirement | Version |
|---|---|
| Python | 3.10+ |
| Node.js | 18+ |
| npm | 9+ |
| Docker + Compose | Any recent version (Docker option only) |
